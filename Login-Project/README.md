<h1>Project Login 🖥️</h1>
<p>A simple HTML & CSS project, the project it’s meant to show a login form (without backend, only the front-end). You can see the responsiveness in the project where it goes from desktop view to mobile view.</p>


<h2>📜 Technologies</h2>
<p>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>VS Code (for writing the code)</li>
</ul>
</p>

<h2>📁 Features</h2>
<p>
<ul>
  <li>Responsiveness, adapts to different screen sizes</li>
  <li>Easy to read code</li>
</ul>
</p>

<h2>📍The Process</h2>
<p>I was presented with this challenge by my instructor and I wanted to create using what we’ve learned in class. I was trying to create this minimalist login page but the biggest challenge was responsiveness which after many tries I was finally able to do it, it’s far from being perfect but nonetheless the process of learning is what made it fun.</p>

<h2>🚦Running the Project</h2>
<p>
  <ul>
    <li>Clone the repository</li>
    <li>Open in your editor of choice (I use VS Code)</li>
    <li>Run it with live server</li>
  </ul>
</p>
